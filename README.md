# GGR472 Lab 1: Geospatial data for the web
This repository contains code for a webpage with a link to a GeoJSON file, as required for Lab 1

Webpage URL: https://abordachan.github.io/lab1/

## Key repository Contents
- `data/Tennis Courts Facilities - 4326.geojson`: Data file containing point locations of Tennis Courts in Toronto 
- `index.html`: HTML file to render homepage
- `report.html`: HTML file containing report
- `script.js`: JavaScript file to enable button interactivity within homepage
